# Under Bar Lighting #

Software to control an LED strip under the bar.

__Components__

 1. ESP32 Mini
 2. WS2814 LED Strip (50LEDS?)
 3. 128x64 OLED Display (SPI or I2C)
 4. Capacitive touch pad
 5. 5v 3A supply
 6. 3D printed case

### What does it do?
Various lighting effects such as Comet, Marquee, Bounce, et al that display on a WS2812 LED strip that is attatced under the bar top. There is a 128x32 OLED incorporated to display FPS, Current Effect,  Power used, IP Address as that is helpful, and also because I like OLED displays.

* OTA updates
* Web UI for Control
* BLE UI for Control
* MQTT implementation
* Home Assistant Integration
